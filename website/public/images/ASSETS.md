# Image Assets Required

Place the following images in this folder before launch.
All images should be PNG, exported at 2x resolution.

| File | Dimensions | Description |
|------|-----------|-------------|
| `phone-hero.png` | 400×800px | Alarm-active screen — yellow background, mascot demanding, rep counter |
| `phone-motion-gate.png` | 320×640px | Push-up gate — camera view with rep counter overlay |
| `phone-barcode-gate.png` | 320×640px | Barcode scan gate — camera viewfinder aimed at item |
| `phone-voice-gate.png` | 320×640px | Voice gate — microphone waveform + live transcript |
| `mascot-sleepy.png` | 120×120px | Mascot Day 0 state (pixel art) |
| `mascot-pumped.png` | 120×120px | Mascot Day 21 state (pixel art) |
| `mascot-legendary.png` | 120×120px | Mascot Day 30 state (pixel art) |
| `og-image.png` | 1200×630px | Open Graph social share image |

Until real assets are available, the components render placeholder UI with emoji.
